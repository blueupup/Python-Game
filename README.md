 Mushroom Game (Python)

A 2D action‑adventure game created in Python.  
Explore, fight mobs, collect weapons, and manage stats in a custom game world built from scratch using Python and Pygame.

# Project Overview
This project demonstrates backing‑end logic, game loop mechanics, sprite management, and environment interaction. It showcases ability to build a playable application that reflects modular design and object‑oriented principles.

# Key Features
- Character control: movement, stats, inventory  
- Weapon system with melee hitboxes  
- Multiple map files for level variation  
- Enemy system with custom stats  
- Particle effects  
- Modular code structure
